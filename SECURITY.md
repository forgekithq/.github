# Security Policy

## Supported Versions

We support the latest stable release of each product. Older versions do not receive security patches.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Email:** [forgekithq@proton.me](mailto:forgekithq@proton.me)

You can also use [GitHub's private vulnerability reporting](https://github.com/forgekithq/.github/security/advisories/new) to submit directly.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Response

- We will acknowledge your report within **48 hours**
- We will provide a fix or mitigation plan within **14 days** for critical issues
- We will credit you in the release notes (unless you prefer to remain anonymous)

**Please do not open a public issue for security vulnerabilities.**
